# ScaleUp Digital

A performance-driven digital marketing agency website showcasing data-driven growth strategies, conversion optimization, and analytics expertise.

## About

ScaleUp Digital partners with growth-stage brands to increase revenue through:
- **Performance Marketing** — ROI-focused paid media across search and social
- **Conversion Rate Optimization** — Continuous testing to improve visitor-to-customer conversion
- **Analytics & Data** — Measurement frameworks and dashboards for growth visibility

## Features

- Responsive design optimized for all devices
- Service showcase with real case study examples
- Free growth audit request form
- Performance-first approach with measurable outcomes
- Accessibility-focused HTML structure
- SEO-optimized with structured data (JSON-LD)

## Getting Started

1. Clone or download the repository
2. Open `index.html` in a browser, or serve locally:
   ```bash
   npx http-server
   ```
3. Deploy to GitHub Pages or your hosting platform

## Files

- `index.html` — Main page structure
- `styles.css` — Responsive styling
- `main.js` — Navigation interactivity
- `sitemap.xml` — SEO sitemap
- `robots.txt` — Search engine crawling rules
